# Upcoming events

This page shows upcoming events

- **All** - shows events from all the cummunities.
- **Following** - show events only from prayer spaces you are following.
- **Nearby** - shows events only near location you have provided.

## Filter

- **Categopry** - filter events to this category
- **Audience** - filter events to this audience
- **Location**
  - **Physical** - events with physical address
  - **Virtual** - online events
