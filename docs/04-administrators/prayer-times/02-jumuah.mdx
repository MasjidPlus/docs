# Jumuah

We understand that some Masjids or Islamic centers have more than one jumuah, so our forms allows you to add as many jumuah times as you need to.

- select adhan start time
- select iqamah time
- click ![icon](/static/icons/add.svg#icon) to add

To delete Jumuah

- click ![icon](/static/icons/trash.svg#icon) to delete
